## 🛠️ Getting Started

1. **Clone & Install**:
   ```bash
   git clone https://github.com/hydraahmad1234567890-ship-it/J.A.R.V.I.S-Smart-PC-Assistant.git
   cd J.A.R.V.I.S-Smart-PC-Assistant
   pip install -r requirements.txt
   ```

2. **Configure Environment**:
   Create a `.env` file and add:
   ```env
   OPENROUTER_API_KEY=your_key_here
   JARVIS_WEB_TOKEN=StarkIndustries2026
   ```

3. **Launch the Master Server**:
   ```bash
   python server.py
   ```

4. **Access the UI**:
   Visit [http://localhost:5000](http://localhost:5000)

## 📱 Mobile Access
Access the interface on your phone by using your PC's local IP address (e.g., `http://192.168.1.X:5000`) while on the same network.
